This is the intentionally buggy product repo used by the support-agent-demo. A Cursor cloud agent clones this repository from GitHub, finds the planted bug, and opens a fix PR.

Planted bug

Applying a coupon increases the cart total instead of decreasing it. The sign error lives in src/cart.ts inside applyCoupon.

Customer complaint: "When I apply my coupon code, my total goes up instead of down."
